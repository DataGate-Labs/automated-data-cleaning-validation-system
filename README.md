# Automated Data Cleaning and Validation System

> A production-ready backend system for automated data profiling, cleaning, validation, anomaly detection, and quality reporting.

---

## Project Overview

The Automated Data Cleaning and Validation System (ADQVS) is designed to automate the complete data quality pipeline for structured datasets.

Instead of manually inspecting and cleaning datasets, the system performs the following tasks automatically:

- Data Profiling
- Data Cleaning
- Data Validation
- Anomaly Detection
- Report Generation

The project follows a modular architecture, making it easy to maintain and extend.

---

## Features

- CSV, Excel, and JSON support
- Automated data profiling
- Missing value detection
- Duplicate detection
- Data cleaning
- Validation rules
- Statistical anomaly detection
- JSON report generation
- Modular architecture
